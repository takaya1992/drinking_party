# Enjoy drinking!

## USAGE

### 1. Perl環境

#### Perl環境

飛ばす

#### cpanm & carton が動く環境を作る

plenvの場合

```bash
$ plenv install-cpanm
```

perlbrewの場合

```bash
$ perlbrew install-cpanm
```

Carton

```bash
$ cpanm Carton
```

### 2.サーバ動かす

cloneしてくる

```bash
$ git clone https://github.com/takaya1992/drinking_party.git
```

セットアップする。`script/install.sh`すれば、`carton install`とDB・テーブルの作成をしてくれる。
